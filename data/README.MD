# Dados

O dataset usado neste projeto (H&M E-commerce Products, ~105 mil registros)
não está versionado neste repositório devido ao tamanho do arquivo.

Fonte original: https://huggingface.co/datasets/Qdrant/hm_ecommerce_products

Para reproduzir: baixe o parquet da fonte acima e execute o script
scripts/converter.py para gerar o CSV usado nas etapas seguintes.
